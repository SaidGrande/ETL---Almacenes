# Proyecto: ETL - Almacenes de Datos

## Descripción
Este proyecto implementa un proceso **ETL (Extract, Transform, Load)** para la gestión y almacenamiento de datos en un entorno de almacenes de datos. Se utilizan herramientas y librerías clave para la extracción, transformación y carga de datos en una base de datos relacional.

## Instalación
Para ejecutar el proyecto, simplemente descarga los archivos del repositorio y ejecuta el siguiente comando en la terminal para instalar las dependencias necesarias:

```
pip install pandas psycopg2
```

## Tecnologías Utilizadas
- **Python**: Lenguaje de programación principal.
- **Pandas**: Para la manipulación y análisis de datos.

## Uso
Una vez instaladas las dependencias, el proyecto se puede ejecutar desde la terminal con:

```
python hotel_main2.py
```

Este proceso ejecutará la lógica ETL definida en el script, extrayendo los datos, procesándolos y cargándolos en la base de datos.
