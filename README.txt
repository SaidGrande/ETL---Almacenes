===============================================
        PROYECTO: ETL - ALMACENES DE DATOS
===============================================

📌 DESCRIPCIÓN:
Este proyecto implementa un proceso **ETL (Extract, Transform, Load)** 
para la gestión y almacenamiento de datos en un entorno de almacenes 
de datos. Se utilizan herramientas clave para la extracción, 
transformación y carga de datos en una base de datos relacional.

-----------------------------------------------
🚀 INSTALACIÓN:
Para ejecutar el proyecto, simplemente descarga los archivos del 
repositorio y ejecuta el siguiente comando en la terminal:

    pip install pandas psycopg2

-----------------------------------------------
🛠️ TECNOLOGÍAS UTILIZADAS:
- Python 🐍 → Lenguaje de programación principal.
- Pandas 📊 → Manipulación y análisis de datos.
